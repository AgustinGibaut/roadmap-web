¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que usamos para estructurar el contenido de las páginas web. Define títulos, párrafos, enlaces, imágenes y más.

✨ Ejemplo: 


<h1>Hola mundo</h1>
<p>Bienvenido a mi sitio web</p>

¿Qué es CSS?

CSS (Cascading Style Sheets) se usa para aplicar estilos a los elementos HTML. Define colores, fuentes, espaciados, animaciones y más.

🎨 Ejemplo: 


h1 {
  color: blue;
  font-size: 2rem;
}

¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite añadir interactividad a tu sitio web. Es el lenguaje que hace que las páginas sean dinámicas.

🚀 Ejemplo básico: 


const saludo = () => {
  alert('Hola mundo desde JS!');
};
saludo();

¿Qué es Bootstrap?

Bootstrap es un framework de CSS que facilita el diseño de sitios web responsivos. Viene con componentes listos como botones, cards y grids.

🧩 Ejemplo de botón: 


<button class="btn btn-primary">Haz clic aquí</button>
📦 Agregá su CDN: 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

¿Qué es Tailwind CSS?

Tailwind CSS es un framework de utilidades para construir interfaces modernas. En lugar de escribir CSS personalizado, usás clases ya definidas para diseñar directamente en el HTML.

⚡ Ejemplo: 


<button class="bg-blue-500 text-white px-4 py-2 rounded">Click aquí</button>

¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite llevar un seguimiento de los cambios en el código fuente. Es esencial para trabajar en equipo y evitar conflictos.

📌 Comandos básicos:

  • git init: Inicializa un repositorio
  • git add .: Agrega los archivos al área de staging
  • git commit -m "mensaje": Crea un snapshot del proyecto
  • git push: Sube los cambios al repositorio remoto

¿Qué es GitHub?

GitHub es una plataforma en línea que permite alojar tus repositorios Git y colaborar con otros desarrolladores. Es muy utilizada para proyectos open source y gestión de código.

🌍 URL de ejemplo: https://github.com/tuusuario
